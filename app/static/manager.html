Power BI и выгрузка

Сайт не может по клику автоматически запустить Power BI на вашем ПК — это ограничение браузера и безопасности Windows.

Как быстрее всего: скачайте CSV → откройте Power BI Desktop → Получить данные → Текстовый/CSV → укажите файл → разделитель ;, кодировка UTF-8.

Дополнительно: «почти в один клик» в браузере

В Chrome или Edge после скачивания нажмите на стрелку у файла в панели загрузок и при необходимости включите «Всегда открывать файлы такого типа». По умолчанию CSV чаще открывается в Excel, а не в Power BI — для Power BI удобнее один раз сохранить шаблон отчёта (.pbix), подключённый к папке «Загрузки», и открывать его, затем нажимать Обновить после новой выгрузки.
